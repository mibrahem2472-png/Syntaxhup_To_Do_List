# Syntaxhup_To_Do_List
Smart ToDo is a modern, responsive task manager built with HTML, CSS, and JavaScript. Add, delete, and complete tasks, track progress with a dashboard, and save tasks in localStorage. Sleek frosted glass UI with smooth animations.
A modern, responsive, and interactive To-Do list app built with **HTML, CSS, and JavaScript**.

## Features
- Add, delete, and toggle tasks
- LocalStorage persistence
- Dashboard with total, completed, and progress percentage
- Responsive design for desktop and mobile
- Frosted glass UI with hover animations

## Usage
1. Open `index.html` in your browser.
2. Add tasks using the input box.
3. Click on the checkbox to mark as done.
4. Click ❌ to delete tasks.
